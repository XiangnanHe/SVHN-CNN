There are four .py files: dataloader.py, reg_proposer.py, datatrainer.py, and datadetector.py, based on Python2
To run the dataloader, you will need to download and unzip the train and test data from SVHN online repository into folder "datasets"
To run the data trainer, you will need to run the dataloader first.
To run the run_v2.py, simply unzip the assignment zip file, and go to the folder, type in "python run_v2.py" under Python2 environment
